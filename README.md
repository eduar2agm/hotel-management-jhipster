# hotel-management-jhipster
Sistema de Gestión Hotelera con JHipster, React y Keycloak, seguro y escalable. Soporta roles de admin, empleado y cliente con RBAC. Incluye gestión administrativa, operaciones de recepción y portal de clientes. Modelo de dominio en JDL, con buenas prácticas y documentación para despliegue local.
